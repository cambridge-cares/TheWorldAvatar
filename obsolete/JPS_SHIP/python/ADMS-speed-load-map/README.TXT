Prerequisites:

1. Install Python3.6 (use 64bit package! https://www.python.org/downloads/release/python-368/)
2. Install virtualenv (https://virtualenv.pypa.io/en/stable/)
3. Create virtual environment: virtualenv -p [path to python3.6] env
4. activate virtual environment (https://virtualenv.pypa.io/en/stable/userguide/#activate-script)
5. install required depencdencies:  pip install -r requirements.txt
6. run 'python ADMS-Map-SpeedTorque-NOxSoot.py' or 'SpeedLoadMap.bat' (on MS Windows)
